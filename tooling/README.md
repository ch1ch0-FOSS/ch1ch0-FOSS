# Tools: Maintained Forks & Contributions

Custom configurations and patches for core open-source tools. Each directory contains upstream source + your customizations.

## Core Tools Maintained
- **zk/** - Zettelkasten CLI with custom features
- **lazygit/** - Git TUI with custom keybindings
- **sway/** - Wayland window manager patches
- **nvim-plugins/** - Custom Neovim plugins

## Workflow
1. Upstream repository in each directory
2. `custom/` or `patches/` subdirectory for your modifications
3. Upstream sync strategy documented in each tool's README

## Contributing Back
See each tool's CONTRIBUTING section for upstream contribution workflow.
