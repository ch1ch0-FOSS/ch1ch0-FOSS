# Contributing

This is a portfolio/documentation repository showcasing professional infrastructure and development practices.

While not accepting external contributions, this repository follows open-source best practices for transparency, maintainability, and learning.

## Standards
- Markdown lint: GitHub Actions validation
- Shell scripts: shellcheck validation
- Documentation: Complete, no TODOs in public view
- Commits: Conventional Commits format
