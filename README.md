## Infrastructure Philosophy 

**Core Mission: Complete Operational Ownership**

Building infrastructure for total local control and agency—free from vendor lock-in and
corporate dependency. Every service is on-premise, self-hosted, and operator-controlled.

**Unix Philosophy in Practice:**

Modular tools that do ONE thing well, completely owned:
    * Syncthing → P2P file sync (no central authority)
    * Vaultwarden → Password management (your data, your control)
    * Forgejo → Git version control (GitHub alternative, full operator ownership)
    * Zk → Zettelkasten knowledge management (local-rst, Markdown-based)

Each service is independent, interoperable, and completely operator-owned. Zero SaaS
dependency, zero vendor lock-in, zero data extraction.
